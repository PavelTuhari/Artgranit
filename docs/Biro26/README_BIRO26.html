Biro26 — Nomenclator / Listă de prețuri / Import

RO: Modul pentru importul mărfurilor din fluxul BIRO26_GOODS în nomenclatorul ERP TMS_UNIVERS și construirea listelor de prețuri (OfficePlus). RU: Модуль импорта товаров из фида BIRO26_GOODS в справочник ERP TMS_UNIVERS и построения прайс-листов (OfficePlus). Фазы ТЗ 1–5.

1. База данных и подключение (важно) / DB & connection

OfficePlus = Oracle 11g (11.2.0.4, charset CL8MSWIN1251). python-oracledb работает с 11g только в thick-режиме (Instant Client). Thick — переключатель уровня всего процесса и ломает основное thin+wallet подключение Artgranit к Oracle Cloud (production nufarul.eminescu.md). Поэтому Biro26 обращается к officeplus через изолированный subprocess-воркер; основное приложение остаётся thin и не затрагивается.

2. Объекты БД / DB objects

Существующие в officeplus (используются как есть)

  • BIRO26_GOODS — входной фид (~78 000 строк).
  • TMS_UNIVERS (TIP='P' товары, 'O' организации), TMS_MPT — карточка.
  • VPR01M_GROUPS, VPR1D_PRDATE, VTPR1D_PERPRLIST (view + INSTEAD OF), TPR1D_PERPRLIST.
  • TMS_ORG (поставщики; имя — через TMS_UNIVERS TIP='O'), TMS_SYSGR (категории-корни).
  • YBIRO_Import_Marfa — PL/SQL пакет (вся бизнес-логика импорта).

Новые (префикс YBIRO_) — деплоятся отдельно на officeplus

  • YBIRO_MAP_PROFILE + YBIRO_MAP_PROFILE_SEQ + триггер — профили маппинга.
  • YBIRO_MAP_PARAM — значения переменных пакета g_* (1 строка на параметр).

Деплой: ./venv/bin/python deploy_biro26_app_tables.py (идемпотентно) или sqlplus officeplus/<pwd>@orange.una.md:4024/cloudbd.world @sql/biro26/01_biro26_app_tables.sql. Это не входит в deploy_oracle_objects.py (та БД — облачная wallet-БД Artgranit).
