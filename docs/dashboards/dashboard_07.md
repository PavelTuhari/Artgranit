# Dashboard 07 — Кредиты Iute

Интеграция с Iute API для оформления кредитов через MyIute.

## Виджеты

- **Оформление кредита Iute** — Check Auth → Create Order → Check Status
- **Документация Dashboard** — Описание виджетов и DDL/DML
- **Output · Логи** — Логи операций Iute

## API

- `GET /api/credit-iute/check-auth` — Проверка авторизации
- `POST /api/credit-iute/create-order` — Создание заказа
- `GET /api/credit-iute/order-status` — Статус заказа
