# Dashboard 06 — Кредитный портфель Бомба

Отдельный режим для администратора кредитного портфеля в виде **пивота**:

- **По вертикали:** группы (категории) и товары
- **По горизонтали:** кредитные организации (банки) и кредитные предложения (программы)

## Возможности

- Фильтрация по категориям (множественный выбор) и по названию/артикулу товара
- Ячейки: ✓ — программа доступна для категории, — — нет
- **Настройки:** форма связей «товарные группы × кредитные предложения» (матрица категория × программа), фильтр по банку

## API

- `GET /api/credit-admin/pivot/meta` — категории, банки с программами
- `GET /api/credit-admin/pivot/matrix` — матрица категория × программа
- `GET /api/credit-admin/pivot/products?category_ids=1,2&search=...&limit=500` — товары с фильтрами
- `GET/POST /api/credit-admin/matrix` — матрица для настроек (как в админке 04)
