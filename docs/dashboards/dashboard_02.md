# Dashboard 02: Custom Dashboard 02

## Описание

Дополнительный кастомный дашборд с расширенными возможностями для разработки и тестирования.

## Назначение

Этот дашборд предназначен для:
- **Разработчиков**: Тестирование новых виджетов
- **QA инженеров**: Проверка функциональности
- **Администраторов**: Кастомные метрики и мониторинг

## Доступные виджеты

Аналогично Dashboard 01, включает все базовые метрики и Custom SQL виджеты.

## Развертывание

См. инструкции в [Dashboard 01](./dashboard_01.md).

## Доработка

Этот дашборд можно использовать как шаблон для создания новых кастомных дашбордов.

### Создание нового дашборда на основе этого

1. **Скопируйте JSON конфигурацию:**
```bash
cp dashboards/dashboard_02.json dashboards/dashboard_06.json
```

2. **Измените dashboard_id:**
```json
{
  "dashboard_id": "06",
  "dashboard_name": "My Custom Dashboard"
}
```

3. **Настройте виджеты:**
Добавьте или удалите виджеты по необходимости.

4. **Создайте документацию:**
Создайте `docs/dashboards/dashboard_06.md` с описанием.
