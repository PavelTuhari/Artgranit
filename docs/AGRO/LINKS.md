# AGRO — Legături rapide

Local (portul implicit 3003): înlocuiți `localhost:3003` cu host-ul/portul dvs. dacă e cazul.
Server remote: înlocuiți `92.5.3.187:8000` cu domeniul/host-ul dvs.

---

### Documente tipărite AGRO

- **URL general:** `/UNA.md/orasldev/agro-document/<id>?type=<type>`
  Tipuri disponibile: `purchase_act`, `weight_ticket`, `shipping_note`, `invoice`, `export_decl`, `qa_protocol`, `gmp_checklist`, `haccp_report`, `mass_balance`
