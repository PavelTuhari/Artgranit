Публично доступные требования к приемке фруктов
Kaufland / Metro / Linella (Молдова)

Назначение документа: рабочая справка для поставщика и отдела качества. Это не заменяет договор, приложение по качеству или индивидуальную спецификацию сети.

Как читать документ. Ниже сведены только публично доступные требования и рыночные ориентиры. Для Kaufland публичная база наиболее подробная. Для Metro есть публичные требования по прослеживаемости/холодовой цепи и опубликованный групповой benchmark по остаткам пестицидов. Для Linella в открытом доступе есть главным образом общие требования по свежести, качеству и возврату несоответствующей продукции.

1) Базовый нормативный минимум (ЕС / Молдова)

Для свежих фруктов (включая яблоки) базовый минимум приемки строится на маркетинговых стандартах ЕС и национальной гармонизации Молдовы. Практически это означает:

  • товар должен быть целым, здоровым, чистым, без вредителей и без нетипичной внешней влаги;
  • товар должен быть пригодным к продаже и транспортабельным;
  • должна быть указана страна происхождения;
  • должны соблюдаться требования к маркировке, сортировке и однородности партии.
Блок Что обычно проверяют на приемке Что это значит для поставщика
Класс качества Extra / I / II, внешний вид, допустимые дефекты, однородность Для сетей фактически безопасный минимум — Class I, если договором не согласовано иное
Калибр Средний размер, доля плодов ниже минимального калибра, смешение размеров Партия должна быть равномерной; минимальный калибр сети обычно строже общего рыночного допуска
Зрелость Brix, твердость, окраска Нужен стабильный eating quality, а не только внешний вид
Документы Прослеживаемость, лаборатория, фитосанитарные / health документы, origin Документы должны совпадать с маркировкой партии и лота
Логистика Температура, упаковка, паллетизация, состояние тары Срыв холодовой цепи и поврежденная упаковка часто ведут к отказу или уценке

2) Сравнение сетей

Сеть Что можно уверенно подтвердить по публичным материалам Что это значит при приемке яблок и других фруктов Уровень уверенности
Kaufland Moldova
  • Действуют общие требования GSR для поставщиков, обязательные по всей цепочке поставки.
  • Система охватывает и Республику Молдова.
  • Высокая прослеживаемость до поля / производителя.
  • Инспекции, сертификации, pre-harvest release / blocking.
  • Публично заявлена внутренняя политика по остаткам пестицидов строже закона; часть действующих веществ исключена полностью.
  • Ожидайте Class I или выше по свежим фруктам, аккуратную сортировку и высокую однородность.
  • Партия должна быть полностью прослеживаемой: лот, производитель, происхождение, дата/код.
  • Наличие лабораторного контроля и прозрачности supply chain критично.
  • Даже при формальном соответствии закону партия может не пройти внутренние спецификации сети.
 Высокий по процессу; средний по числовым порогам для конкретного SKU (в открытом доступе обычно не публикуются)
METRO Moldova
  • Публично заявлены: verified chain, сертифицированное качество, проверенные поставщики, ежедневные проверки соответствия и прослеживаемости, строгий температурный контроль и холодильная логистика.
  • В групповых публичных benchmark-материалах по produce фигурируют требования: GlobalG.A.P., лимиты по остаткам пестицидов, ограничение числа действующих веществ, лабораторный контроль и supplier audits.
  • На приемке обычно критичны: temperature compliance, traceability, документы по безопасности, качество упаковки и маркировки.
  • Для produce-модели целесообразно считать базовым профилем Class I + холодовая цепь + полная прослеживаемость.
  • Если в контракте нет отдельного annex, можно стартовать с группового benchmark и затем заменить его официальным приложением METRO Moldova.
 Высокий по процессу; средний по числовым лимитам именно для METRO Moldova
Linella
  • Публично декларируются: свежесть, качество, безопасные отношения с domestic producers, verified suppliers.
  • В публичном интервью руководства сети прямо говорится, что несоответствующие quality standard партии быстро возвращаются и поставщики не должны грузить Class II.
  • Для практической модели приемки безопасно исходить из Class I, нормальной однородности партии, корректной упаковки и маркировки.
  • При отсутствии отдельного annex рекомендуется письменно согласовать: минимальный калибр, допуски по дефектам, Brix, температуру и правило по скидке/сортировке.
 Средний: общие требования подтверждаются, но детальный публичный numeric supplier annex не найден

3) Практический перевод в критерии приемки

Kaufland — рабочий профиль приемки

  • Класс: не ниже Class I.
  • Однородность: высокая, без заметного смешения размеров/окраски в коробе.
  • Прослеживаемость: обязательна по лоту и производителю.
  • Документы: лаборатория/безопасность/происхождение должны быть доступны по партии.
  • Пестициды: ориентируйтесь не только на закон, но и на более строгую внутреннюю политику сети.

Примечание: конкретные числовые tolerances по SKU обычно содержатся во внутренних приложениях к договору, а не в открытом доступе.

METRO — рабочий профиль приемки

  • Класс: не ниже Class I.
  • Холодовая цепь: обязательна; отклонение температуры — частая причина претензии.
  • Прослеживаемость: обязательна от поставщика до партии.
  • Безопасность: наличие подтверждения по остаткам и supplier verification желательно включать в базовый пакет.
  • Если нужен числовой pesticide profile: можно использовать групповой benchmark как временный рабочий стандарт до получения локального annex.

Linella — рабочий профиль приемки

  • Класс: не ниже Class I.
  • Свежесть: критический фактор; «пограничные» партии лучше не отгружать.
  • Упаковка и маркировка: должны быть чистыми, читаемыми, без пересорта по лотам.
  • Согласование: попросить у сети письменный перечень tolerances по калибру/дефектам и правило по скидке за сортировку.

4) Минимум, который стоит согласовать письменно с любой сетью

Параметр Что согласовать заранее
Коммерческий класс Extra / I / II; отдельно прописать запрет на Class II, если сеть работает только с retail grade
Калибр Минимальный размер, метод измерения, допустимый % below-min
Дефекты % minor defects, % severe defects, правило по гнили/плесени/давлению
Зрелость Brix, твердость, окраска, состояние мякоти
Остатки пестицидов Лимит относительно legal MRL, список запрещенных действующих веществ, число активных веществ
Температура Допустимый диапазон при приемке, время выстаивания, кто и как меряет
Документы CoA / лаборатория, traceability sheet, phytosanitary / health papers, сертификаты поставщика
Уценка / отказ Формула скидки, право на сортировку, фото-акт, срок подачи рекламации
Рекомендация. Используйте этот документ как pre-qualification checklist. Перед первой отгрузкой запросите у сети: supplier manual, produce annex, packaging spec, labeling spec, temperature spec и claims procedure.
