1) Базовый нормативный минимум (ЕС / Молдова)

Для свежих фруктов (включая яблоки) базовый минимум приёмки строится на маркетинговых стандартах ЕС и национальной гармонизации Молдовы. Практически это означает:

  • товар должен быть целым, здоровым, чистым, без вредителей и без нетипичной внешней влаги;
  • товар должен быть пригодным к продаже и транспортабельным;
  • должна быть указана страна происхождения;
  • должны соблюдаться требования к маркировке, сортировке и однородности партии.
Блок Что обычно проверяют на приёмке Что это значит для DIPP FRUCT
Класс качества Extra / I / II, внешний вид, допустимые дефекты, однородность Поставлять не ниже Class I; Class II — только по отдельному письменному соглашению
Калибр Средний размер, доля плодов ниже минимального калибра, смешение размеров Партия должна быть равномерной; сортировка на линии DIPP перед отгрузкой
Зрелость Brix, твёрдость, окраска Контроль рефрактометром и пенетрометром на складе DIPP перед отгрузкой
Документы Прослеживаемость, лаборатория, фитосанитарные / health документы, origin Traceability sheet по каждому лоту: сад, дата сбора, обработки, лаборатория
Логистика Температура, упаковка, паллетизация, состояние тары Отгрузка из холодильника DIPP при 0…+4 °C; рефрижератор обязателен

2) Сравнение сетей

Сеть Что можно уверенно подтвердить по публичным материалам Что это значит при приёмке яблок и других фруктов Уровень уверенности
Kaufland Moldova
  • Действуют общие требования GSR для поставщиков, обязательные по всей цепочке поставки.
  • Система охватывает и Республику Молдова.
  • Высокая прослеживаемость до поля / производителя.
  • Инспекции, сертификации, pre-harvest release / blocking.
  • Публично заявлена внутренняя политика по остаткам пестицидов строже закона; часть действующих веществ исключена полностью.
  • Ожидайте Class I или выше по свежим фруктам, аккуратную сортировку и высокую однородность.
  • Партия должна быть полностью прослеживаемой: лот, производитель, происхождение, дата/код.
  • Наличие лабораторного контроля и прозрачности supply chain критично.
  • Даже при формальном соответствии закону партия может не пройти внутренние спецификации сети.
 Высокий по процессу; средний по числовым порогам для конкретного SKU
METRO Moldova
  • Публично заявлены: verified chain, сертифицированное качество, ISO 22000, HACCP, ежедневные проверки.
  • В групповых benchmark-материалах по produce: GlobalG.A.P., лимиты по остаткам пестицидов, supplier audits.
  • На приёмке критичны: temperature compliance, traceability, документы по безопасности, качество упаковки.
  • Базовый профиль: Class I + холодовая цепь + полная прослеживаемость.
 Высокий по процессу; средний по числовым лимитам
Linella
  • Публично декларируются: свежесть, качество, verified suppliers, ~150 магазинов.
  • В интервью FreshPlaza: несоответствующие quality standard партии быстро возвращаются, Class II запрещён.
  • Исходить из Class I, нормальной однородности партии, корректной упаковки и маркировки.
  • Письменно согласовать tolerances по калибру/дефектам и правило по скидке/сортировке.
 Средний: общие требования подтверждаются, публичный numeric annex не найден

3) Рабочие профили приёмки по сетям

Kaufland — рабочий профиль

  • Класс: не ниже Class I.
  • Однородность: высокая, без заметного смешения размеров/окраски в коробе.
  • Прослеживаемость: обязательна по лоту и производителю.
  • Документы: лаборатория/безопасность/происхождение должны быть доступны по партии.
  • Пестициды: ориентируйтесь не только на закон, но и на более строгую внутреннюю политику сети.

METRO — рабочий профиль

  • Класс: не ниже Class I.
  • Холодовая цепь: обязательна; отклонение температуры — частая причина претензии.
  • Прослеживаемость: обязательна от поставщика до партии.
  • Безопасность: подтверждение по остаткам и supplier verification в базовый пакет.

Linella — рабочий профиль

  • Класс: не ниже Class I.
  • Свежесть: критический фактор; «пограничные» партии лучше не отгружать.
  • Упаковка и маркировка: должны быть чистыми, читаемыми, без пересорта по лотам.

5) Шкала кодирования дефектов (DIPP Defect Coding D01–D27)

Как пользоваться. При приёмке партии инспектор проверяет выборку (min 5 коробов или 2% партии) и фиксирует каждый обнаруженный дефект кодом из таблицы ниже. Коды D01–D27 синхронизированы с системой скоринга AGRO модуля (17-точечная взвешенная оценка). Профили приёмки настраиваются в Admin → Acceptance Profiles для каждой торговой сети.

5.1 Критические дефекты (D01–D09) — автоматический отказ при обнаружении

Код Дефект / Defect Проверка AGRO Вес Описание / пример
D01 Гниль / Rot SERIOUS_OK 15 Любой очаг гниения, мягкая зона с неприятным запахом
D02 Плесень / Mould SERIOUS_OK 15 Видимый мицелий, споры, грибковые пятна
D03 Живые вредители / Live pests SERIOUS_OK 15 Насекомые, личинки, ходы. Фрукт с ходом = заражён. Любое обнаружение = отказ
D04 Посторонний запах / Foreign odour SERIOUS_OK 15 Химический, гнилостный, нетипичный запах. Любое обнаружение = отказ
D05 Несоответствие классу / Class mismatch CLASS_OK 10 Поставлен ниже заявленного коммерческого класса (Extra / I / II)
D06 Температура нарушена / Temp breach TEMP_OK 10 Температура при приёмке выше допустимого диапазона (0…+4 °C)
D07 Нет лаб. отчёта / Missing lab report LAB_OK 8 Отсутствует лабораторный анализ по партии (CoA)
D08 Нет фитосанитарного серт. / Missing phyto PHYTO_OK 4 Отсутствует фитосанитарный / health сертификат
D09 Нет traceability / Missing traceability TRACE_OK 6 Невозможно установить происхождение: поле, дата сбора, обработки

5.2 Значительные дефекты (D10–D14) — снижение оценки, возможна сортировка

Код Дефект / Defect Проверка AGRO Вес Описание / пример
D10 Нажимы / Bruising MINOR_OK 8 Потемнение мякоти >1 см², вдавления >3 мм глубиной
D11 Деформация / Deformation MINOR_OK 8 Нехарактерная форма, трещины кожицы, расколы, асимметрия
D12 Ниже мин. калибра / Below min calibre CALIBRE_OK + BELOW_MIN_OK 8+5 Средний калибр ниже нормы или доля плодов below-min > допуска
D13 Смешение размеров / Mixed sizes MIXED_OK 4 Неоднородность размеров в коробе, смешение калибровочных групп
D14 Brix ниже нормы / Low Brix BRIX_OK 5 Содержание сахара ниже минимального порога для сорта

D15–D19 зарезервированы для будущих дефектов категории «Значительный».

Код Дефект / Defect Проверка AGRO Вес Описание / пример
D20 Пов. дефекты кожицы / Skin defects MINOR_OK 8 Царапины, руссетинг, пятна — поверхностные, без повреждения мякоти
D21 Следы листьев / веточек / Leaves & twigs MINOR_OK 8 Посторонние включения в коробе: листья, веточки, земля
D22 Неполная маркировка / Incomplete label LABEL_OK 2 Отсутствуют обязательные поля на этикетке короба (см. раздел 8)
D23 Нарушение упаковки / Pack damage PACK_OK 3 Повреждённая, деформированная или грязная тара
D24 Residue single > MRL SINGLE_RESIDUE_OK 2 Одно действующее вещество превышает максимально допустимый уровень (MRL)
D25 Residue total > limit TOTAL_RESIDUE_OK 2 Суммарные остатки пестицидов превышают групповой порог
D26 Glyphosate > MRL GLYPHOSATE_OK 1 Глифосат обнаружен выше допустимого уровня
D27 Кол-во акт. веществ > лимит / Active substances ACTIVES_OK 2 Число обнаруженных действующих веществ превышает допустимый лимит

5.4 Решение по партии (синхронизировано с AGRO Scoring Engine)

ПРИНЯТЬ (ACCEPT) — Все 6 критических проверок пройдены И взвешенная оценка ≥ 95 из 100.
Формула: сумма весов всех пройденных проверок (max 95) + оценка свежести (0–5) = итоговый балл.
СОРТИРОВКА / СКИДКА (ACCEPT_WITH_SORTING) — Все 6 критических проверок пройдены И оценка ≥ 85 (настраиваемый порог accept_min_score в профиле сети). Партия принимается с условием пересортировки или ценовой скидки.
ОТКАЗ (REJECT) — Любая критическая проверка не пройдена (D01–D09) ИЛИ оценка < 85.

7.2 Требования к таре

  • Тара чистая, сухая, без посторонних запахов, без повреждений (разрывов, проколов, деформаций).
  • Новый картон — для каждой поставки (не переиспользованный, если нет IFCO).
  • Внутренняя подложка / вкладыш из пищевого материала (не газета, не техническая бумага).
  • Крышка (для телескопа) — плотно закрывается, но не раздавливает верхний слой.
  • Обязательны вентиляционные отверстия (min 5% площади стенки) для поддержания холодовой цепи.

7.3 Паллетизация

Параметр Требование
Паллета EUR 1200 × 800 мм (стандарт) или CHEP. Паллета чистая, без сломанных досок
Укладка Коробки 600×400: 2 в ряд × 2 в глубину = 4 короба на слой
Высота паллеты Max 180 см (с паллетой); 10–12 слоёв в зависимости от высоты короба
Фиксация Стрейч-плёнка + 2 стреппинг-ленты. Верхний слой не должен свисать за край паллеты
Паллетная этикетка SSCC (Serial Shipping Container Code) или минимум: поставщик, продукт, сорт, класс, калибр, лот, дата упаковки, кол-во коробов, нетто
Температура при погрузке 0…+4 °C (core temperature). Зафиксировать в CMR / delivery note
