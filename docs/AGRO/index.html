Standarde de achiziție NOU

  • Cerințe Kaufland / Metro / Linella pentru recepția fructelor Cerințe publice de acceptare, profiluri de recepție pe rețea, criterii de calitate, temperatură, proslediabilitate. Referință pentru departamentul QA.
  • Șablon specificație tehnică furnizor fructe Formular de pre-aprobare: clasă comercială, calibru, defecte, maturitate, Brix, ambalare, logistică, reguli de recepție. Anexă la contract.
  • Model de acceptare fructe Moldova (XLSX) Tabel Excel cu parametri de recepție, toleranțe pe produs, formule de calcul. Vizualizare online în browser.

Module UI

  • AGRO MDI Shell Interfață principală Interfața principală cu tab-uri pentru toate modulele AGRO — punct de intrare unificat
  • AGRO Admin Administrare Date de referință (furnizori, clienți, depozite, articole), configurare sistem, rapoarte și export XLSX/CSV
  • AGRO Recepție (Field) Operator câmp Achiziții de la producători, cântărire lăzi, generare și scanare coduri de bare — interfață optimizată pentru tabletă
  • AGRO Depozit (Warehouse) Operator depozit Gestionarea stocurilor și mișcărilor, monitorizare temperatură în celule frigorifice, sarcini depozit, alerte automate
  • AGRO Calitate / HACCP (QA) Inspector calitate Liste de verificare GMP/HACCP, controale de calitate pe loturi, blocări automate la neconformitate critică, planuri HACCP și înregistrări CCP
  • AGRO Vânzări (Sales) Operator vânzări Documente de vânzare, alocare loturi FIFO, expedieri la export, declarații vamale

Obiecte Oracle

  • 36 tabele normalizate + 5 vizualizări Prefix AGRO_ — master data, operațiuni, vânzări, calitate/HACCP, audit. 36 triggere auto-ID + 3 triggere audit.
