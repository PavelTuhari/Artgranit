Colass — материалы ТЗ и бизнес-процессы

Материалы использованы для модуля Colass (каталог F3/F5 и сметчик): бизнес-процессы, схема потоков документов, справочники и демо-офферта по строительству газопровода.

Бизнес-процессы Colass

RACI, Mermaid-диаграммы, роли, артефакты, чек-листы.

Схема бизнес-процессов KRONOS

Диаграмма: заказ, смета F3/F5/F7, снабжение, списание материалов.

Каталог Colass

Иерархия работ и ресурсные нормы расхода (F5 + F3).

Сметчик Colass

Подбор работ/ресурсов и формирование оферты с итогами по типам.

Colass CRM

Классическая воронка: лид -> квалификация -> смета -> переговоры -> регистрация контракта.

Colass Contracts

Реестр договоров (мастер, контакты, маршруты, позиции из сметы), связан с CRM и сметчиком.

CRM бизнес-процесс

Схема процесса с источником лидов из Email (IMAP intake).
