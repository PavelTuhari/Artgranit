1. Назначение проекта

Веб-приложение для работы с Oracle Autonomous Database (OCI) и бизнес-модулями:

  • Oracle SQL Developer–подобный режим — выполнение SQL, браузер объектов (таблицы, представления, процедуры, пакеты).
  • Dashboard — метрики в реальном времени (CPU, RAM, сессии), Top SQL, tablespaces (WebSockets).
  • Кредиты — админка программ/банков/матрицы, оператор оформления заявок, интеграции EasyCredit и Iute, портфель «Бомба», отчёты.
  • Nufarul — приём заказов (химчистка, пакеты): админка услуг/статусов (RU/RO/EN), интерфейс оператора (создание заказа, поиск по штрихкоду), автоматическая генерация первичных документов: Bon de Comandă, Comandă (заказ), Jurnal Registru (журнал регистраций) — печать/просмотр по заказу или список последних заказов.
  • Shell — мультипроектный вход (shell, gara, credit, nufarul) с дашбордами по slug.
  • Документация — просмотр Markdown/HTML, ТЗ Nufarul, XLSX/DOC/PDF viewer, галерея JPG с OCR-таблицами.

Архитектура: MVC. Backend: Python + Flask. Real-time: Flask-SocketIO. БД: oracledb (Oracle Wallet).

4. Конфигурация (config.py, .env)

Все секреты и пути — в .env. config.py читает через os.environ.get() и load_dotenv().

Основные переменные (.env)

ПеременнаяОписание
DB_USER, DB_PASSWORDПользователь и пароль Oracle
WALLET_DIRПуть к распакованной папке Oracle Wallet
WALLET_PASSWORDПароль кошелька (если ewallet.pem)
CONNECT_STRINGTNS connect string (из tnsnames.ora)
ENVIRONMENTLOCAL | REMOTE
PORTПорт сервера (3003 локально, 8000 удалённо)
SECRET_KEYFlask session secret
DEFAULT_USERNAME, DEFAULT_PASSWORDЛогин/пароль веб-интерфейса
EASYCREDIT_*Sandbox/production, base_url, api_user, api_password
IUTE_*base_url, api_key, pos_identifier, salesman_identifier
RATELIMIT_*Лимиты запросов (по умолчанию 200/hour, memory://)

Переопределения API (без секретов в репо): data/easycredit_settings.json, data/iute_settings.json. Методы Config.easycredit_base_url(), Config.iute_api_key() и т.д. учитывают эти файлы.

Языки: BABEL_DEFAULT_LOCALE = 'ru', SUPPORTED_LANGUAGES = ['ru', 'ro', 'en'].

6. Модели (database.py)

models/database.py

  • DatabaseConnection — проверка wallet (WALLET_DIR), get_connection(), get_pool() (oracledb pool, min=1, max=5), поддержка ewallet.pem и config_dir.
  • DatabaseModel — контекстный менеджер для запросов; методы для выполнения SQL и выборок (используются контроллерами).

Подключение только через .env: DB_USER, DB_PASSWORD, CONNECT_STRING, WALLET_DIR, WALLET_PASSWORD.

13. Запуск (локально и на сервере)

Локально (Mac)

cd /Users/paveltuhari/cursorsprojects/OCI/Artgranit
cp .env.example .env   # заполнить WALLET_DIR, пароли и т.д.
./run_local.sh
# или: source venv/bin/activate && export ENVIRONMENT=LOCAL PORT=3003 && python3 app.py

Приложение: http://localhost:3003. Логин — DEFAULT_USERNAME / DEFAULT_PASSWORD из .env.

Удалённый сервер

cd /home/ubuntu/oracle_test_app   # или путь проекта на сервере
source venv/bin/activate
export ENVIRONMENT=REMOTE PORT=8000
nohup python3 app.py > app.log 2>&1 &

Или ./full_restart.sh. Доступ: http://<SERVER_IP>:8000.
