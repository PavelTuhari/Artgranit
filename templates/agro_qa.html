📖 Инструкция / Instructiuni de utilizare ▶
  1. В верхней таблице — список всех лабораторных проверок. Кликните на строку для просмотра содержимого анализа.
  2. Используйте поле поиска для фильтрации по партии, чек-листу или инспектору.
  3. В нижней панели отображаются пункты выбранной проверки: параметр, норма, фактическое значение, вердикт (ОК / FAIL).
  4. Цветовая индикация: зелёный = соответствует, красный = не соответствует.
  5. Кнопка + Проверка открывает форму для создания новой инспекции.
  6. Критические параметры помечены звёздочкой *.
