📖 Инструкция / Instructiuni de utilizare ▶
  1. В верхней таблице отображается список документов закупок (шапки). Кликните на строку для просмотра содержимого.
  2. Используйте вкладки статусов (Все / Черновик / Подтверждено / Закрыт) для фильтрации.
  3. Поле поиска фильтрует по номеру, поставщику и дате.
  4. В нижней панели отображаются строки выбранного документа: товар, вес, цена, сумма.
  5. Для редактирования документа нажмите кнопку «Открыть» в нижней панели — откроется форма редактирования.
  6. Кнопка + Новая закупка создаёт новый документ приёмки.
